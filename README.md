# Overview

FinSight AI is an intelligent personal finance analytics platform that transforms raw bank transaction data into actionable financial insights.

The platform automatically analyzes income, expenses, savings, and spending behavior while generating personalized financial recommendations using Google's Gemini AI. Users can upload financial data, visualize spending trends, monitor KPIs, and interact with an AI financial advisor through natural language.
